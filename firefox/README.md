# SoundThing Tracker

> [!CAUTION]
> Modern Browser Security doesn't allow extensions to send their own web requests to whatever domains they please. For this reason, consider this bare minimum proof-of-concept a terrible idea. This version will never be officially published.

To add this extension to Firefox, open `about:debugging` and select the "This Firefox" tab. Then click on "Load Temporary Add-on" and select the `manifest.json` file in this directory.
