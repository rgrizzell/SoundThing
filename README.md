# SoundThing
Pairs with a Firefox extension to show what's playing in SoundCloud on your CircuitPython device.

Currently only supports the LILYGO T-Display S3.

![A LILYGO T-Display-S3 displaying the Artist title and track alongside the album artwork.](/doc/assets/soundthing.png)

## Device
Load the contents from the `src` directory to your device.
Add wifi credentials to `settings.toml`

## Extension
See [firefox/README.md](firefox/README.md)
